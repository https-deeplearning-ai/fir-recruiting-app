# Archived Test Files

This directory contains exploratory and diagnostic test scripts that were used during development but are no longer actively maintained or necessary for ongoing development.

## Categories

### Debug/Diagnostic Tests
- `test_*_debug.py` - Ad-hoc debugging scripts
- `test_*_diagnostic.py` - One-time diagnostic tools
- `test_*_manual.py` - Manual testing scripts

### Exploratory Tests
- Early prototypes and experiments
- API endpoint exploration scripts
- Feature spike tests

## Note
These files are preserved for historical reference but are not required for:
- Development workflow
- CI/CD pipeline
- Production deployment

For active tests, see the main test suite documentation in CLAUDE.md.
